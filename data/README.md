# SRE-Gym Data Directory

This directory contains incident data used by the SRE-Gym environment.

## Files

- `real_incidents.json` — Downloaded from logpai/loghub (generated by prepare_datasets.py)
- `metadata.json` — Dataset provenance and statistics

## Generating Data

```bash
# Download real incident data
python scripts/prepare_datasets.py

# With custom sample count
python scripts/prepare_datasets.py --samples 1000
```

## Dataset Sources

| Dataset | Source | Description |
|---|---|---|
| logpai/loghub | HuggingFace | Real system logs from 30+ production systems |

## Fallback

If dataset download fails (no internet / missing token), the script
automatically generates realistic synthetic incidents as fallback.
The core task JSON files (task1_easy.json etc.) are always available.
